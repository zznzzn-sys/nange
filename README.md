# 土豆兄弟 - 2D肉鸽游戏

一个基于HTML5 Canvas开发的2D肉鸽（Roguelike）游戏，玩家控制土豆角色在各种波次的敌人攻击中生存。

## 项目概述

- **游戏类型**: 2D肉鸽生存游戏
- **技术栈**: HTML5 Canvas + JavaScript + CSS3
- **开发状态**: 进行中
- **目标平台**: Web浏览器（桌面和移动设备）

## 游戏特性

### 核心玩法
- 波次生存模式，每波敌人数量逐渐增加
- 玩家可以购买升级提升能力
- 随机生成的敌人和物品
- 永久死亡机制（Roguelike元素）

### 玩家系统
- 生命值管理
- 攻击力、攻速、移动速度等属性
- 多弹道射击系统
- 爆炸范围攻击
- 金钱系统和商店购买

### 敌人系统
- 不同类型的敌人AI
- 追逐、攻击、死亡效果
- 敌人波次生成系统

### 视觉效果
- 粒子系统（射击、爆炸、死亡效果）
- 流畅的动画和过渡效果
- 响应式UI设计
- 现代化玻璃拟态设计风格

### 音效系统
- Web Audio API实现的丰富音效
- 射击、爆炸、受伤、死亡等音效
- 背景音乐系统
- 界面交互音效

## 技术实现

### 游戏引擎
- 基于HTML5 Canvas的自研游戏引擎
- 60FPS游戏循环
- 高性能渲染和碰撞检测

### 物理系统
- 简单的2D物理模拟
- 碰撞检测算法
- 粒子物理效果

### UI系统
- 响应式游戏界面
- 状态栏和属性面板
- 移动设备触摸控制

### 音效系统
- 高级Web Audio API实现
- 多音效叠加播放
- 音效分类和参数化控制

## 开发指南

### 运行游戏
```bash
# 进入游戏目录
cd potato_bros

# 启动本地服务器
python -m http.server 3000

# 访问游戏
http://localhost:3000
```

### 开发环境
- 现代Web浏览器（Chrome、Firefox、Safari等）
- 支持ES6+ JavaScript
- 本地HTTP服务器（用于音效播放）

### 代码规范
- 使用ES6+ JavaScript语法
- 遵循游戏开发最佳实践
- 代码注释和文档齐全
- 性能优化和内存管理
