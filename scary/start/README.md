# 恐怖校园 - 网页解密游戏

## 游戏概述
一款基于React + Phaser.js的恐怖解密网页游戏，玩家扮演调查员在校园中调查一系列恐怖事件，揭开真相。

## 游戏特色
- 🎮 **悬疑剧情** - 反转不断，出人意料却又在意料之中
- 🏫 **校园背景** - 真实的校园环境，恐怖氛围营造
- 🔍 **探索解密** - 收集线索，推理真相
- 🎵 **恐怖音效** - 沉浸式恐怖体验

## 技术栈
- **前端框架**: React 18 + TypeScript
- **游戏引擎**: Phaser.js 3
- **状态管理**: Zustand
- **路由管理**: React Router
- **音效管理**: Howler.js
- **构建工具**: Vite

## 游戏玩法
1. **角色扮演**: 玩家作为调查员进入校园
2. **场景探索**: 在不同场景中寻找线索
3. **对话互动**: 与NPC对话获取信息
4. **物品收集**: 收集关键物品推进剧情
5. **真相揭露**: 通过推理找出事件真相

## 剧情大纲
在一个看似普通的校园中，发生了一系列离奇的恐怖事件。玩家作为调查员，需要深入校园，与各种角色互动，收集线索，最终揭开隐藏在校园背后的惊人真相。
