# 恐怖校园游戏 - 开发指南

## 项目概述
这是一个基于React + Phaser.js的恐怖解密网页游戏，玩家在校园中调查恐怖事件真相。

## 技术架构
- **前端框架**: React 18 + TypeScript
- **游戏引擎**: Phaser.js 3
- **状态管理**: Zustand
- **构建工具**: Vite

## 项目结构
```
scary/start/
├── src/
│   ├── components/     # React组件
│   │   ├── HomePage.tsx    # 首页
│   │   ├── GameScene.tsx   # 游戏场景
│   │   └── *.css          # 样式文件
│   ├── scenes/         # Phaser游戏场景
│   │   └── GameScene.ts    # 主游戏场景
│   ├── stores/         # 状态管理
│   │   ├── gameStore.ts    # 游戏状态
│   │   └── GameProvider.tsx # 状态提供者
│   ├── data/          # 游戏数据
│   │   └── gameData.ts     # 剧情和场景数据
│   ├── utils/         # 工具函数
│   │   └── soundManager.ts # 音效管理
│   ├── App.tsx        # 主应用组件
│   ├── main.tsx       # 应用入口
│   └── *.css          # 全局样式
├── public/            # 静态资源
│   └── assets/       # 游戏资源
├── package.json       # 项目配置
├── vite.config.ts    # Vite配置
└── README.md         # 项目说明
```

## 快速开始

### 1. 安装依赖
```bash
cd scary/start
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 访问游戏
打开浏览器访问 `http://localhost:3000`

## 开发说明

### 添加新场景
1. 在 `src/data/gameData.ts` 中添加场景配置
2. 在 `src/scenes/` 中创建对应的Phaser场景
3. 更新游戏状态管理

### 添加新NPC
1. 在场景配置中添加NPC数据
2. 创建对应的对话内容
3. 实现交互逻辑

### 资源制作指南
- 图片尺寸: 推荐800x600像素
- 音效格式: MP3或WAV
- 文件命名: 使用英文小写和下划线

## 剧情大纲
游戏分为三个章节：
1. **初入校园** - 了解基本情况
2. **学生证词** - 收集关键线索  
3. **真相大白** - 揭开校园秘密

## 技术注意事项
- Phaser场景与React组件通过状态管理通信
- 游戏资源需要放置在public/assets目录
- 使用TypeScript确保类型安全
- 响应式设计支持移动端
